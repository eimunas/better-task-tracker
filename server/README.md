## Description

Better Task Tracker application.

## Project setup

```bash
$ npm install
```

## Compile and run the project

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Build docker compose

```
$ docker compose up --build -d
```

## Run tests

```bash
# unit tests
$ npm run test

# e2e testss
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
